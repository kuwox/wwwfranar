##############################
VirtueMart eCommerce Bundle
based on the official
Joomla! 1.5.22 [senu takaa ama woi]

##############################

This Package is a modified version of the original Joomla! distribution
available at http://www.joomla.org.
NOTE: This is not an official Joomla! release - it will not be supported or warranted by the Joomla! Project or Open Source Matters. 
		All support questions should be addressed to http://forum.virtuemart.net.
		
	
It has the following features / modifications:

* The complete VirtueMart e-commerce Application (Version 1.1.8) is pre-installed
  - 1 Component
  - 1 Main Module (already published)
  - 10 additional Shop modules
  - 2 Plugins for VirtueMart (1 for Search, 1 for Content - both published)

* eXtplorer 2.1.0 RC3 is already installed
* CustomQuickIcons Component is installed and configured to show a quick link to VirtueMart
* JCE 1.5.7 Mambot is pre-installed and the default WYSIWYG Editor
* Moneybookers Payment Gateway Component is pre-installed 
* the following Templates are included:
  - JA Larix by the Joomlart.com Template Club (for Terms of Usage and Licensing Details see /templates/ja_larix/LICENSE.php)
  - Jamba by JoomlaShack.com (GNU/GPL Template)
  
- - - - - - - - - - - - - - - - - - - - - - - - - - - -
For Installation please refer to the original Joomla! INSTALL file
in this directory!
Note: If you select "Install Sample Data" during Joomla Installation,
the installer will insert Sample Data for VirtueMart as well!

PLEASE CARE FOR YOUR DATABASE. IF YOU DON'T SELECT
"Backup existing tables" all your existing tables
WILL BE DROPPED.
