Custom QuickIcons - Management

Latest Version
CQI exists of 2 parts: this component and the module.
The module can be published at 2 positions:
  1. icon (replaces the original module quickicons and is standard)
  2. cpanel (is then visible with other modules at the right side)
An additional way of publishing is, to copy the module and publish both (at icon and cpanel), but with diferent icons and access levels.
Toolbar Management
  • Activate
    Activates denoted items to be accessable.
    You will get the same effect by clicking on the icon in the row 'Published'
  • Deactivate
    Deactivates denoted items to be inaccessable.
    You will get the same effect by clicking on the icon in the row 'Published'
  • New
    Creates a new item - see also Toolbar Manage
  • Edit
    Edits the denoted item
    You will get the same effect by clicking on the name
  • Copy
    Copies denoted entries
    The copy(ies) will have a note [Copy of] as well only accessable for the Super Admin
  • Delete
    Deletes denoted items
    Deleting is permanent!
  • Edit CSS
    With this function you can edit the CQI own CSS.
    Activation has to be done in the module
