Custom QuickIcons - Overview

Latest Version
CQI exists of 2 parts: this component and the module.
The module can be published at 2 positions:
  1. icon (replaces the original module quickicons and is standard)
  2. cpanel (is then visible with other modules at the right side)
An additional way of publishing is, to copy the module and publish both (at icon and cpanel), but with diferent icons and access levels.

Menutabs
Settings can be defined in following menus:
  • General
    • Target
      Define here the target (link) for the menuitem (after index2.php) - will be automatically finished by the system
      If external links shall be called only absolute Links are allowed (like: http://www.link.xxx)
    • New Window
      Shall the link be opened in a new window
    • Activated
      Is this menu published?
    • Order
      New menus will be automatically at first position (not changeable!).
      After saving the reordering can be made
    • Access
      Define here which usergroups have access to this menuitem - access levels can be redefinded
      Only usergroups with access to the backend are visible.
      Warning: After installing this component, all menuitems have access only for the Superadmin, which means only the menuitems are only visible and clickable for the Superadmin
    • User
      You can define this icon to only one (1) user, it will be usable further only for this user
      Attention: This setting has priority against the group!
  • Text
    • Prefix
      Each menuitem can be defined with a text prefix (bold, italian, underline, etc.
    • Text
      Text as it will appear within the module. Shorter is better! Additional text can be defined under Title
    • Predefined formats for the text
      Bold, Italian, Underline
      Every combination of valid HTML.tags can be used
    • Suffix
      Final suffix for the text
      Warning: Wrong settings can block the display of the menu item!
    • Shortcut
      For users with disabilities a shortcut (or access key) should be defined. To be used also instead of clicking the menu item with the mouse
      Shortcuts will be called by the keyboard with ALT + [character]
      These shortcuts must be unique - avoid to define 2 menus with the same shortcut
    • Title
      Text which will be shown if you hoover over the menuitem You can define additional text with a description to the menuitem - not longer than 60 chars
  • View
    • View as
      3 settings are possible:
      • Icon & Text (Standard)
      • Only Icon
      • Only Text
    • Imagefolder
      Here you can see all image folder found in your CMS as selectable dropdown
    • Icon
      Choose which icon shall be displayed (opens new window)
      With the dropdown Imagefolder you can choose the directory.
      Tip: use the Mediamanager to create an own directory with your icons
      Attention: If you called the mediamanager to create (and upload new images), you have to call CQI egain from the menu otherwise you will not see the new directory in the dropdown list!
  • Check
    • Check Component
      If a component is deleted, with this check an error can be avoided.
      If set to Yes, before the link (icon) is displayed the component will be checked if valid and installed
    • Name & Path
      If previous setting is set to yes, define here the internal path to this component, e.g. customquickicons/admin.custumquickicons.php will be finished by the system to:
      ../administrator/components/com_customquickicons/admin.customquickicons.php
